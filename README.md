### framework-base

基础工具包(Jdk1.8)

- redis 封装工具类
- cache 本地缓存封装工具类
- Result 结果集
- CommonConstants 常用常量类
- utils 常用字符串、时间日期、uuid、json等封装工具类


引入

    <dependency>
        <groupId>com.lul</groupId>
        <artifactId>framework-base</artifactId>
        <version>1.0.0-SNAPSHOT</version>
    </dependency>
